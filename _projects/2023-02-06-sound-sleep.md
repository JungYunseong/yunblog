# Project Title
### 최고의 숙면을 경험하세요, Sound Sleep
***
## Key Features
* 수면 단계 인식
  - AI모델을 통해 실시간으로 측정된 사운드를 통해 수면 단계를 측정합니다.
* 수면 단계별 맞춤형 사운드 제공
  - 측정된 수면 단계별 맞춤형 바이너리 비츠 사운드를 생성하여 최적화된 숙면을 유도합니다.
* 스마트 알람
  - 자연스러운 기상을 유도하는 스마트 알람이 제공됩니다.
